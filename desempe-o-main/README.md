    #  prueba-de-desempeño
    

    🎸 Black Sabbath - Fan Page Project

    This is a tribute website dedicated to the legendary heavy metal pioneers, Black Sabbath. The project was built as a practical exercise in responsive web design, focusing on clean code and a "dark metal" aesthetic.

     Features

    Dark Aesthetic: A custom-themed UI inspired by the band’s iconic style.
    Event Management: A detailed tour schedule table featuring dates, locations, and ticket pricing.
    Optimized Media Gallery: A responsive CSS Grid layout specifically tuned for square images (640x640px).
    Fully Responsive: Includes a mobile-friendly hamburger menu for seamless browsing on all devices.
    Smooth Navigation: Implements smooth-scroll behavior for internal section links.

    Tech Stack

    HTML5: Semantic structure for better SEO and accessibility.
    CSS3: Custom styling using Flexbox, CSS Grid, and CSS Variables for easy theme management.
    JavaScript (Vanilla): Client-side logic for the mobile menu toggle and automatic footer year updates.

    Project Structure
    /
    ├── index.html      # Main site structure
    ├── styles.css      # Custom styles and responsive layouts
    ├── main.js        # Logic for interactivity and UI effects
    └── imagenes/       # Assets folder for band photos (640x640) and logos

    Setup & Usage

    Clone or download this repository.
    Place your images in the /imagenes/ directory (recommended size: 640x640px).
    Open index.html in any modern web browser to view the site.

    Design Specification

    Color Palette:
    Primary Background: #0a0a0a (Deep Black)
    Accent Color: #6a1b9a (Sabbath Purple)
    Image Handling: The gallery uses the aspect-ratio: 1/1 and object-fit: cover properties to ensure all photos look consistent and professional regardless of their original dimensions.
